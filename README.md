# SpellRise - GitHub Source Mirror

Este repositório mantém apenas código-fonte, configs, scripts e documentação.

## Assets e Builds
- Assets pesados (Content) não são versionados no GitHub.
- Builds de Cliente/Servidor são publicados no itch.io.

## Estrutura
- `Source/` C++
- `Config/` configurações do projeto
- `Scripts/` automações de build/teste
- `Fontes/` documentação de design/arquitetura

## Observação
Para desenvolver localmente, mantenha uma cópia dos assets no workspace local.
